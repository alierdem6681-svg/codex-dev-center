# LIVING DOCUMENTATION POLICY

Bu sistemde hiçbir geliştirme paketi dokümantasyon, state, rapor, log ve dashboard kayıtları güncellenmeden tamamlanmış sayılmaz.

Her paket sonunda ilgili dosyalar güncel tutulur:

- docs/AGENT_ONBOARDING_MAP.md
- AGENTS.md
- constitution/ANAYASA.md
- docs/HANDOVER.md
- docs/ROADMAP.md
- memory/project_memory.md
- state/system_state.json
- state/module_registry.json
- state/module_settings.json
- state/action_catalog.json
- reports/
- logs/

Zorunlu minimum paket bitirme kuralı:

1. HANDOVER güncellenecek
2. ROADMAP güncellenecek
3. project_memory güncellenecek
4. system_state güncellenecek
5. reports/ altında rapor oluşacak
6. logs/system.log yazılacak
7. logs/audit.log yazılacak
8. Yeni modül/servis varsa AGENT_ONBOARDING_MAP.md güncellenecek
9. Kullanıcı kalıcı kural verdiyse ANAYASA.md güncellenecek
